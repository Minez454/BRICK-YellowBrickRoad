## 🌐 How to Show People

### Option 1: Local Demo (Fastest)
```bash
# Navigate to demo folder
cd demo

# Start a simple HTTP server
python -m http.server 8000
# OR
npx serve .

# Open in browser
# Visit http://localhost:8000
```

### Option 2: GitHub Pages (Free & Easy)
```bash
# 1. Create a new GitHub repository
# 2. Upload the demo folder contents
# 3. Go to repository Settings → Pages
# 4. Select "Deploy from a branch"
# 5. Choose main branch and /docs folder
# 6. Save and wait 2-3 minutes

# Your demo will be live at:
# https://yourusername.github.io/your-repo/
```

### Option 3: Netlify Drop (Super Easy)
```bash
# 1. Drag the demo folder into netlify.com
# 2. Your site is instantly live!
# 3. You get a free URL like: yellowbrickroad-demo.netlify.app
```

### Option 4: Vercel (Professional)
```bash
# 1. Install Vercel CLI
npm i -g vercel

# 2. Deploy from demo folder
cd demo
vercel --prod

# 3. Your demo is live at a vercel.app URL
```

### Option 5: Firebase Hosting (Google)
```bash
# 1. Install Firebase CLI
npm install -g firebase-tools

# 2. Initialize Firebase
firebase init

# 3. Deploy
firebase deploy

# 4. Live at your-project.web.app
```
